# PrehistoricWorld
An Attempt to implement a simple 2d survival game written in C as a back-end layer over SDL 2 library to interface with Haskell, while using Haskell to implement the game logic.

* To build:
```bash
make all
```
* To run:
```bash
cd $Project_folder
cd out
./PrehistoricWorld
```
* To test:
```bash
cd $Project_folder
cd out
./test
```
